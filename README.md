# NeoData 专业金融终端

一款功能强大的金融数据查询工具，支持多种市场数据实时查询和专业技术分析。

## ✨ 功能特性

### 📊 数据查询
- **股票行情**：支持 A股、港股、美股实时行情查询
- **大盘指数**：上证指数、深证成指、创业板、沪深300等
- **板块数据**：半导体、人工智能、新能源等40+热门板块
- **贵金属**：黄金、白银、铂金、钯金实时行情
- **外汇**：美元/人民币、欧元/人民币等主要汇率

### 📈 技术分析
- 均线系统（5日、10日、20日、60日）
- MACD指标（金叉/死叉信号）
- RSI相对强弱指数
- 布林带分析

### ⚡ 买卖信号
- 综合技术指标判断
- 买入/卖出信号提示
- 目标价和止损价建议

### 🕐 交易时间查询
- 各市场交易时间一览
- 当前交易状态智能判断
- 下一交易日提示

## 🚀 使用方法

```bash
# 查询股票
python query.py --query "贵州茅台"

# 查询大盘指数
python query.py --query "上证指数"

# 查询交易时间
python query.py --query "交易时间"

# 查询全部数据
python query.py --query "全部"
